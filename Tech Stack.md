Выбранные инструменты обеспечивают высокую скорость разработки MVP и легкость последующего масштабирования.

Backend:

FastAPI (Python): Высокая производительность, асинхронность для работы с Telegram-ботом и автоматическая генерация документации API (Swagger).

SQLAlchemy / Alembic: Для управления базой данных и миграциями.

Pydantic: Валидация данных на входе.

Frontend (Dashboard):

React + Vite: Быстрая сборка и высокая отзывчивость интерфейса.

Tailwind CSS: Для создания адаптивного интерфейса, удобного для планшетов.

dnd-kit / react-beautiful-dnd: Реализация плавного перемещения карточек.

FullCalendar / DayPilot: Библиотека для реализации сетки календаря с режимами (день, неделя, месяц, год).

Database & Real-time:

PostgreSQL: Основное хранилище данных.

WebSockets: Для мгновенной синхронизации данных между ботом и панелью менеджера.

Infrastructure:

Docker & Docker Compose: Контейнеризация для быстрой развертки на любом VPS.
